Deep Q Learning Doom -- Ongoing

Implementing a rational agent using Deep Q-Learning to play the original Doom video game developed by iD Software. Understanding the vast game state space of Doom, we decided to utilize a neural network that estimated Q-Values of these complex game states which inherently provided us with staggering results. Throughout the project, we utilized a variety of technologies such as Nvidia CUDA Parrellization, TensorFlow, and PyTorch.
